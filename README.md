# Gastroenterology Practice Tracker

Local-only iPhone-friendly clinic and endoscopy tracker.

## Storage
- Records are stored in this browser using localStorage.
- Automatic local snapshots are created whenever data is saved.
- Google Drive is used only for backup and restore, not live synchronization.

## Important
Before replacing website files or clearing Safari website data, download a JSON backup or confirm a Google Drive backup exists.
